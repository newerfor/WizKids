WizKids - Личный помощник воспитателя
📱 Мобильное приложение для воспитателей, педагогов и репетиторов, заменяющее бумажные журналы и разрозненные заметки.

🏗 Архитектура проекта
Модульная структура — код организован по фичам, обеспечивая независимость, переиспользуемость и более быструю сборку

Single Activity — вся навигация построена на одном Activity с использованием Jetpack Compose Navigation, что даёт единый lifecycle и упрощает управление состояниями

Clean Architecture с разделением на слои (data, domain, presentation)

MVVM, Repository Pattern, Use Cases

🛠 Технологический стек
Kotlin, Jetpack Compose, Material 3

Room (локальная БД с TypeConverters и миграциями)

Koin (DI), Gson

Coroutines, StateFlow

📦 Фичи и возможности
👥 Управление детьми (профили, фото, документы, этапы развития)

💰 Финансовый учёт (баланс, оплаты, стоимость занятий)

📅 Интеллектуальный календарь с визуализацией посещений

🔍 Поиск, фильтрация, сортировка по разным параметрам

👤 Профиль специалиста с экспортом в PDF

🌗 адаптивный дизайн, плавные анимации

✅ Полная обработка ошибок и состояний загрузки

🔧 Установка
Требования: Android SDK 34+, Kotlin 1.9+, Gradle 8.0+
